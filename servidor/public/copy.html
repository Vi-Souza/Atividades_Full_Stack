Introdução

C++ (em português: lê-se "cê mais mais", em inglês lê-se see plus plus) é uma linguagem de programação compilada multi-paradigma (seu suporte inclui linguagem imperativa, orientada a objetos e genérica) e de uso geral. Desde os anos 1990 é uma das linguagens comerciais mais populares, sendo bastante utilizada na criação de sistemas operacionais e de jogos.

Java é uma linguagem de programação orientada a objetos e desenvolvida a partir de uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems. Em 2008 a Java foi adquirida pela empresa Oracle Corporation.

Java é uma linguagem de programação criada por desenvolvedores da SUN Microsystems por volta de 1990, pouco antes da explosão da Internet. Essa linguagem foi muito bem aceita e mantém-se até hoje com uma grande comunidade de usuários. É uma linguagem com sintaxe semelhante ao C++ e orientada a objetos, ainda que com diferenças importantes.
